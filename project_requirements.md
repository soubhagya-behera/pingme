Project Name: PingMe

Description:
A real-time chat application built using Spring Boot WebSocket and React.

Tech Stack:

Backend:
- Java 17
- Spring Boot
- Spring WebSocket
- STOMP
- Spring Security
- MySQL
- Maven

Frontend:
- React.js
- JavaScript
- HTML
- CSS

Features:

1. User Join Chat
2. Real-Time Messaging
3. Private Messaging
4. Online User Tracking
5. Message Timestamps
6. Auto Delete Messages
7. Responsive Design
8. Logout Functionality

Database:
MySQL

Future Features:
- Group Chat
- File Sharing
- Voice Messages

Rules:
- No paid APIs
- Leave API key placeholders empty
- Create .env.example if needed
- Do not hardcode secrets
- Production-ready clean code
